# Gettin started

## Important information:

The backend portion was not written by me. It was generated to create a mock-up of the API and data

Follow these simple instructions to install the dependencies in the backend and frontend projects

### BACKEND

```bash
cd .\backend\
```

```bash
npm i
```

```bash
npm run dev
```

Application should start on PORT 3009

### FRONTEND

```bash
cd .\frontend\
```

```bash
npm i
```

```bash
npm run dev
```

Application should start on PORT 5173
